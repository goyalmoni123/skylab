Introduction

Welcome to the Skylab Developer Guidelines. This document provides instructions and tools to effectively work with Skylab's platform.

Setup Guide

  1. Clone the Skylab repository from GitHub.
  2. Install dependencies: npm install.
  3. Set up the database using MongoDB/PostgreSQL scripts provided in the repo.
  4. Run the server: npm start.
  5. Access Skylab locally at http://localhost:3000.

API Documentation

The Skylab platform provides a RESTful API. Below are some key endpoints:

Endpoint Method Description
/api/register POST Register a new user
/api/jobs GET Fetch all test jobs
/api/feedback POST Submit feedback

Best Practices

  • Follow semantic versioning for all updates.
  • Write clear, concise comments for all new code.
  • Use ESLint and Prettier for consistent coding standards.
  • Ensure test coverage for all new features.
