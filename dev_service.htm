Getting Started as a Developer

Follow these steps to get started on Skylab:

  1. Create an account on the platform and set up your developer profile.
  2. Create test jobs, specifying testing requirements and scope.
  3. Monitor feedback and performance results on the developer dashboard.

Developer Dashboard Overview

The developer dashboard provides key tools for managing tests:

  • Feedback Summary: View insights and detailed comments from testers.
  • Performance Metrics: Track app performance in real-world environments.
  • Test History: Access a log of all your past test jobs and results.

Best Practices for Using Skylab

Maximize the value of Skylab’s platform with these tips:

  • Define clear objectives and guidelines for each test job.
  • Engage with testers by providing constructive feedback on their reports.
  • Utilize performance data to make iterative improvements to your app.
